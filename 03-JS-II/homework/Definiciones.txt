* `&&`, `||`, `!` && este simbolo se utiliza para indicar que ademas debe complirse otra condicion para ser true,


|| se utiliza para indicar que se puede cumplir una condicion o otra para ser true


! y finalmente este simbolo indica que la condicion no se debe cumplir para ser true

true es que se cumple la condicion, false que no se cumple  9 != 10  retorna true      10 != 10 retorna false
